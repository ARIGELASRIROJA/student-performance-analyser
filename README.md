# student-performance-analyser
Java console app to analyze student marks using arrays. Computes max, min, second max/min, average, and counts students above average. Built with optimized O(n) logic and proper edge case handling to strengthen core data structures concepts.

# 🎓 Student Performance Analyzer

A simple Java project that analyzes student marks using arrays.

## 🚀 Features
- Find Maximum marks
- Find Minimum marks
- Find Second Highest marks
- Find Second Lowest marks
- Calculate Average
- Count students above average

## 🧠 Concepts Used
- Arrays
- Loops
- Conditional statements
- Time complexity optimization (O(n))

## 💻 Tech Stack
- Java

## 📌 How to Run
1. Clone the repository
2. Compile the code:
   javac StudentAnalyzer.java
3. Run the program:
   java StudentAnalyzer

## 📊 Sample Input
5  
10 20 30 40 50  

## ✅ Sample Output
Max: 50  
Min: 10  
Second Max: 40  
Second Min: 20  
Average: 30.0  
Students above average: 2
